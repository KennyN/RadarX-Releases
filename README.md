# RadarX Releases

This repository contains official releases of RadarX Desktop Application.

## Download

Visit the [Releases](https://github.com/KennyN/RadarX-Releases/releases) page to download the latest version.

## Installation

1. Download `RadarX Setup.exe` from the latest release
2. Run the installer
3. Follow the on-screen instructions

## System Requirements

- Windows 10 or later (64-bit)
- 4 GB RAM minimum
- 500 MB available disk space
